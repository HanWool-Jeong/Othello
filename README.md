# 오델로 게임

## 개요
컴퓨터와 오델로를 할 수 있는 게임입니다.

컴퓨터는 플레이어가 흑돌을 놓은 뒤 2초 뒤에 놓습니다.

컴퓨터는 플레이어의 흑돌을 많이 얻을 수 있는 방식으로 백돌을 놓습니다.
